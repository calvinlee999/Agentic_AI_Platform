# Images Directory

This directory contains visual assets for the Agentic AI Platform documentation.

## Files

- `agentic_bank_workflow.png` - Diagram showing the Agentic Banker reasoning and execution workflow
- `fsi_use_cases.png` - Overview of top use cases across Banking, Capital Markets, and Insurance sub-verticals

## Adding Images

To add the provided images to this repository:

1. Save the images from your presentation/document
2. Place them in this `/images` directory
3. Ensure filenames match the references in the documentation:
   - Agentic Bank workflow diagram → `agentic_bank_workflow.png`
   - FSI Use Cases by sub-vertical → `fsi_use_cases.png`

## Image Guidelines

- Use PNG format for diagrams and screenshots
- Use descriptive filenames in lowercase with underscores
- Optimize images for web (recommended max width: 1200px)
- Include alt text in markdown references for accessibility
